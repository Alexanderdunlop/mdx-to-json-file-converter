# MDX to JSON Converter

A simple web-based tool that converts MDX files to JSON format directly in your browser. No server required and complete privacy as all conversions happen locally in your browser.

## 🚀 Live Demo
Visit [alexanderdunlop.github.io/mdx-to-json-file-converter](https://alexanderdunlop.github.io/mdx-to-json-file-converter)

## ✨ Features
- 🖥️ Browser-based conversion
- 🔒 Private & secure (all processing happens locally)
- 📥 Drag & drop interface
- 📤 Automatic JSON download
- 📝 Frontmatter support
- 🎯 Simple and intuitive UI

## 🔧 Usage
1. Visit the website
2. Drag and drop your MDX file or click to select
3. The converted JSON will automatically:
   - Display on the page
   - Download to your computer
